## Overview
This is a polished frontend implementation suitable for a Frontend Developer take-home:
- React (Vite) app with React Router
- Tailwind CSS for styling and responsive layout
- Components: Auth (mocked), Tasks CRUD UI, Protected routes
- API integration layer (axios); easy to swap base URL
- Build-ready (vite build) and small test example (Vitest + React Testing Library)

## Run locally
1. Install: `npm install`
2. Start dev server: `npm run dev`
3. Build: `npm run build`
4. Preview build: `npm run preview`
